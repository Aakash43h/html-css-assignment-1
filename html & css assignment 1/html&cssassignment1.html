Block vs Inline Elements

Understanding Block and Inline Elements

Block Elements

Block elements are used to structure the content into distinct sections. Examples include <h1>, <p>, and <div>. These elements occupy the full width of their container and start on a new line.

This is a block element (div).

This is a block element (p).

This is a block element (h2).

Inline Elements

Inline elements are used within text and do not start on a new line. Examples include <a>, <i>, and <b>. These elements only take up as much width as their content.
